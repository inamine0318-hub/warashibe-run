# ❄ 氷島の卵 — ICE ISLAND EGG

一人用ターン制サバイバルゲーム。極寒の氷島に墜落した調査隊員として、崩壊する氷上を渡り歩き、卵をはじめとした荷物を回収して飛行機で脱出せよ。

---

## 🎮 遊び方

1. キャラクターを選択
2. 難易度を選択（EASY / NORMAL / HARD）
3. ダイスを振って移動マス数を確定
4. 十字キーで移動
5. 全荷物を回収したら飛行機マスへ戻って脱出！

---

## 🕹 操作方法

| 操作 | 内容 |
|------|------|
| 十字キー | キャラクター移動 |
| ダイスボタン | 移動マス数を決定 |
| スキルボタン | キャラクター固有スキルを発動 |
| ピッケルボタン | 周辺マスを調査 |
| ターン終了 | 移動を終えてターンを終了 |
| 🔋 バッテリー | 防寒スーツエネルギーを回復（3回まで） |

---

## ⚠ ゲームギミック

- **床崩壊マス** ── 踏むと4マス吹き飛ばされる
- **凍結ゾーン** ── 踏むとSE-7%・次ターン移動1固定
- **猛吹雪** ── 崩壊・凍結を3回踏むと永続発動。視界が1マスに制限される
- **暗視ゴーグル** ── 吹雪発生後にのみ拾える。全マップを視認可能になる
- **雪ブロック** ── 通行不可の障害物

---

## 👤 キャラクター

| キャラ | 特徴 | スキル |
|--------|------|--------|
| 隊長 | バランス型 | ピッケル探索（2マス先を調査） |
| 囚人 | 高HP | 障害物跳躍（穴・壁を飛び越える） |
| ロシア兵 | 中HP | 暗視スコープ（2ターン全視界確保・2回） |
| 新人 | 低HP | がむしゃら（追加で2マス移動） |
| 化石ハンター（ベテラン） | 高HP | 不死身の執念（HP0で1度だけ自動復活） |

## 🖥 GitHub Pages で遊ぶ

1. このリポジトリをフォーク or クローン
2. GitHubのリポジトリページで **Settings → Pages**
3. Branch: `main` / フォルダ: `/ (root)` を選択して **Save**
4. 数分後に表示されるURLでブラウザから遊べます

```
https://<your-username>.github.io/warashibe-run/
```
